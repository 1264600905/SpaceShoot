自己用unity制作的飞机大战，实现了
1.jsonlite加载数据，玩家和敌人还有玩家升级方向选择都是从json读取数据。
2.地图每次自动生成。
3.场景异步加载。
4.游戏可以设置玩家摇杆大小，敌人ai难度，游戏画质。每次开启游戏自动读取上一次配置。
5.小地图
6.敌人简易血条
7.伤害计算和管理。
8.UGUI的运用使用。
9.多种敌人飞机和玩家换飞机的操作。
10.敌人的AI
